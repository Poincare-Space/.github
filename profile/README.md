# Poincare Space

> Exploring the future of AI memory systems.

## About Us

We are a team of engineers, researchers, and builders working on **POINCARE**, exploring the future of AI memory Systems.

## The Problem

Current AI agents consume memory inefficiently. Context windows are limited, compaction is a loss, retrieval is expensive, and agents lose critical information between sessions. The memory layer is the bottleneck — not intelligence.

## Our Vision: POINCARE

POINCARE study an alternative memory system designed for autonomous AI agents. We're building:

- **Persistent Memory** — Agents remember across sessions, not just within them
- **Efficient Retrieval** — Semantic search that scales, not brute-force context stuffing
- **Agent-Native Architecture** — Memory designed for agents, not retrofitted from databases
